# Rig Picker

A professional floating Rig Picker for Blender.

## Features

- Floating window
- Character body picker
- Add Selected Controls
- Rename controls
- Delete controls
- Show / Hide controls
- Search controls
- Save picker inside .blend
- Future support for multiple picker pages

---

## Development

This project consists of two parts.

### Blender Addon

Responsible for

- Bone selection
- Show / Hide
- Saving picker
- Communication with UI

### PySide6 UI

Responsible for

- Floating Window
- Character Picker
- Search
- Drag & Drop
- Visual Interface
